How do I create my personal web page using Github and Git Bash?

PacMan Project Go to Eyes Exercise

1. Preparation

  1. Your personal e-mail address.
  2. Download Git Bash.
  3. Sign up for a Github account using your personal e-mail account. Please click on Github.
  4. Download any kind of editor program for web-page design. I strongly recommend Visual Studio Code. Please click on Download Visual Studio Code.
  5. Set up a working directory for web page development on your computer.

2. Design web page using an editor program.

  1. Create index.html and save it in your working directory. When someone visits your website, the index.html page is invoked the first time. Therefore, index.html is like a gate in the house.
  2. Prepare your picture and name the picture file as yourname.jpg.

3. Set up a repository on GitHub (Upload pages)

  1. Sign in to GitHub and access your Github account.
  2. Click on "New repository".
  3. Enter yourname.github.io as the repository name. This will be used as your webpage address.
  4. Select the "Public" option for the repository.
  5. Click on the "Create repository" button.
  6. Click on "uploading an existing file" on the next page.
  7. Drag your index.html and yourname.jpg files to the upload window.
  8. Click on the "Commit changes" button.

4. Web page check

  1. Check your website opening using yourname.github.io in the browser.
  2. If you find any bugs, review and provide feedback.

5. Readme file update

  1. Create a log history of release information in the README file.
